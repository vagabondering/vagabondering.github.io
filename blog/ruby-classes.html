  • Instance variables & methods

    • An instance variable, which is defined by the @ prefix works just like a local variable, however the difference is that the instance variable is recognized outside the scope and can used across different methods within the class. You can see in the class example below, @variable is the instance variable and is being used for both initalize and instance_method.

    An instance method is a method that's shared between the instance variables. It's made to function multiple instance variables and not just one specifically.

    def ClassName

    def initialize(argument)
    @variable = argument
    end

    def instance_method
    @variable
    end

    end

    class_instance = ClassName.new(example)
    class_instance.instance_method
