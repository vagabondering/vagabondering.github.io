Instance variables & methods

An instance variable, which is defined by the @ prefix works just like a local variable, however the difference is that the instance variable is recognized outside the method and can used across different methods within the class. The instance variable is available for the methods to perform their action on the same variable. You can see in the class example below, @variable is the instance variable and is being used for the methods, initalize and instance_method.

An instance method is a method that's called on an instance of a class, so it can output different objects as instances from the class. Looking at the sample below, the instance_method is being called on an instance of the class, class_instance.

def ClassName

def initialize(argument)
@variable = argument
end

def instance_method
@variable
end

end

class_instance = ClassName.new(argument)
class_instance.instance_method
