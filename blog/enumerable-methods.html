  • Enumerable Method - #inject


    • The inject method computes a single result value by applying an operation sequentially to all the elements. It takes a value to start with and a block/symbol, and runs that block/symbol for each element in the array returning a sum or collection of the object.
    It's very similar to other enumerable methods like each, map and select. However the difference between inject and each is that inject doesn't operate on each element of the collection separately. And unlike map and select, it doesn't return a modified version of the array.
    The ideal use of inject method is when you want to collect a single result when iterating through the list of values.

    An example of a sum method using inject:

    def sum(array)
    array.inject(0) do |sum, i| sum += i end end

    sum([1,2,3]) #=> 6

    The inject method can also be used to convert and array of arrays to a hash. Here's an example:

    array = [['A', 'a'], ['B', 'b'], ['C', 'c']]
    hash = array.inject({}) do |memo, (key, value)| memo[key] = value memo end

    hash #=> {'A' => 'a', 'B' => 'b', 'C' => 'c'}
