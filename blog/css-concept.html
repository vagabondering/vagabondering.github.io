  • What does it mean to display inline vs using inline block?

    • Both options allow the elements to be next to each other on the same line. Inline block behaves like a block element, where width and length can be set, whereas with inline, those two gets ignored.

  • To use these options, in the css file set the element display to either inline or inline-block. Example: display: inline; or display:inline-block;
