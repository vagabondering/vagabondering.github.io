Ruby Enumerables

each

Executes the given block with each element.

each_with_index

Finds the index position of the array.

each_slice

Breaks the collection up into specified n argument.

select

Selects certain items based on some criteria.

detect/.find

Returns the first element when the block returns true.

reject

Returns the first element when the block returns false.

find_all

Returns array of all elements when the block returns true.

find_index

Returns the index of the first element when the block returns true.

grep

Returns any elements that matches the given argument.

sort

Returns sorted array

sort_by

Returns sorted array in a specific order

max

Returns the highest element

min

Returns the lowest element

map/collect

Keeps the objects when returning an array with the results from block.

inject

Return a single value of all the elements from the given block expression.

count

Returns total number of elements in the collection.

reduce

Reduces the collection down to a single element.

any?

Return boolean true if the block is true for any elements in the collection.

all?

Return boolean true if the block is true for all elements in the collection.

none?

Return boolean true if the block is true for all elements in the collection.
